# UI（Wireframes）

本目录用于描述 #0004 对 UI 的增量改动：在“配置卡片”上增加近 24h 的库存走势（sparkline）。

## 设计要点

- 展示位置：配置卡片（用于“库存监控”与“全部产品”两处）
- 展示形式：作为卡片背景层（background layer）渲染，不额外占用布局空间（不单独占一块“图表区域/标题行”），不挤占正文信息区域
- **尺寸不变**：卡片尺寸/内边距不因走势而改变；不同页面卡片尺寸按 `#0001` 线框图为准：
  - 库存监控页配置卡片：`504x160`
  - 全部产品页配置卡片：`504x208`
  - 云服务器卡片（全部产品页）：`504x104`
- 徽章不拉伸：卡片底部“库存/更新”等徽章使用内容宽度（fit-content），左右 padding 保持紧凑；文字居中以避免右侧大块留白
- 徽章半透明：徽章背景色需半透明（例如 60% 不透明度），保证仍能看到卡片背景走势
- 徽章靠近底部：徽章行靠近卡片底部（保持紧凑底边距），避免卡片底部出现大块空白
- 时间窗：滚动 24 小时（rolling 24h）
- 数据形状：sparse points（每个点带 `tsMinute`），X 轴必须按真实时间比例映射
- 纵轴：固定 0–10；当 `quantity > 10` 时显示为 `10+`（图形截断在顶部并可用样式区分）
- 走势样式：使用“填充折线（area）”（折线 + 半透明填充），而非纯折线
- 无交互：不做 tooltip/缩放/拖拽（避免引入复杂图表库）

- `products-with-trend.svg`：全部产品页（卡片尺寸不变；走势为背景层）
