# UI 预览图（临时）

本目录用于存放由 `ui/*.svg` 自动渲染生成的 PNG 预览图，方便主人在不打开 SVG 的情况下快速查看线框效果。

- 这些 PNG **仅用于临时评审**，不作为项目长期资产；主人确认后可随时删除整个目录。
- 真实交付以 `docs/plan/0001:lazycats-cart-inventory-monitor/ui/*.svg` 为准。
